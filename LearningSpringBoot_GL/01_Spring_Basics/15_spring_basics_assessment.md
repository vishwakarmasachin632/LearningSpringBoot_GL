## 📝 MCQs (30 Questions)

### Q1. Spring Framework ka main purpose kya hai?
A. Database banana  
B. Loose coupling achieve karna  
C. UI design karna  
D. Server banana

✅ **Answer:** B  
**Explanation:** Spring IoC & DI ke through loose coupling provide karta hai.

---

### Q2. IoC ka matlab kya hota hai?
A. Control developer ke paas hota hai  
B. Control JVM ke paas hota hai  
C. Control Spring container ke paas hota hai  
D. Control database ke paas hota hai

✅ **Answer:** C  
**Explanation:** Object lifecycle ka control Spring container manage karta hai.

---

### Q3. Dependency Injection ka fayda kya hai?
A. Fast UI  
B. Tight coupling  
C. Easy testing  
D. Hard maintenance
